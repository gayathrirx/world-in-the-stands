# 🌎 World In The Stands ⚽

AI-curated fan stories from World Cup 2026 — heartwarming, funny, and emotional moments from visitors experiencing the USA for the first time.

Multi-agent pipeline: Scout Agent searches the web → Claude curates, tags, and writes captions → beautiful card feed with two tabs (Fan Stories + Match Buzz).

**Requires `ANTHROPIC_API_KEY` set in Space secrets.**
